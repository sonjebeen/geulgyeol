# 글결 (GeulGyeol)

글결은 같은 한글 문장을 세 번 쓸 때 생기는 움직임 차이를 비교해, 사용자의 필체 개성은 보존하고 가장 불안정한 습관 하나만 교정하는 웹 기반 손글씨 코치입니다.

## 주요 기능

- iPad 손가락, Apple Pencil, 마우스 입력 지원
- 좌표·시간·속도·획 리듬·펜 입력 데이터를 브라우저에서 측정
- 세 번의 필기를 비교한 일관성 점수와 한 가지 집중 교정 제공
- OpenAI API 연결 시 GPT-5.6 비전이 세 필기 이미지를 비교해 문제 글자와 근거를 직접 지목
- 세 필기 오버레이 재생과 교정 전·후 점수 비교
- API 키가 없거나 네트워크가 불안정해도 기기 내 분석으로 계속 동작

## 로컬 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

GPT 코칭을 사용하려면 `.env.example`을 참고해 서버 환경에 `OPENAI_API_KEY`를 설정합니다. 키는 브라우저 코드에 포함되지 않습니다.

## 구조

- `app/page.tsx`: 필기 캔버스, 3회 측정, 분석 결과 UI
- `app/api/analyze/route.ts`: 선택적 OpenAI Responses API 코칭
- `app/globals.css`: iPad 반응형 제품 디자인

원본 획 데이터는 브라우저에서 계산하며 저장하지 않습니다. AI 호출에는 문장과 집계된 분석 수치만 전달합니다.
